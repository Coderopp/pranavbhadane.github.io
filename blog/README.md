# Blog Posts

This directory contains individual blog post pages. Each post follows the same template structure.

## Adding a New Blog Post

1. Copy one of the existing blog post HTML files (e.g., `shipping-fast.html`)
2. Rename it to match your new post title (use kebab-case)
3. Update the content:
   - `<title>` tag
   - Post date
   - Heading (`<h1>`)
   - Main content inside `.post-content`
4. Add the link to the main listing page in `thinking-and-execution.html`
